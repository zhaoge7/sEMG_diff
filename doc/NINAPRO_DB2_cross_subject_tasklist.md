## 7. 实验协议设计

### 7.1 个体内识别 baseline

- [ ] 对每个受试者单独训练和测试。
- [ ] 按 repetition 划分训练集和测试集，避免随机窗口划分。
- [ ] 将该结果作为个体内识别性能上限参考。
- [ ] 报告所有受试者的平均值和标准差。

### 7.2 零校准跨个体识别

- [ ] 实现 leave-one-subject-out，简称 LOSO。
- [ ] 对每一个目标受试者：
  - [ ] 该目标受试者只作为测试集。
  - [ ] 其余 39 名受试者作为源域训练集。
  - [ ] 验证集只能从源域受试者中划分。
  - [ ] 训练阶段不能使用目标受试者标签和目标受试者统计量。
- [ ] 让 40 名受试者轮流作为测试受试者。
- [ ] 报告每名测试受试者结果，以及整体平均值和标准差。
- [ ] 在每个 LOSO fold 中加入稳定学习训练协议。
  - [ ] 只用 39 名源域受试者估计样本权重或稳定约束。
  - [ ] 不使用目标受试者数据估计权重，除非明确声明为无监督域适应。
  - [ ] 保证稳定学习过程不泄漏目标受试者标签。

### 7.3 少样本个体自适应

- [ ] 对每个目标受试者：
  - [ ] 使用其他 39 名受试者预训练模型。
  - [ ] 使用目标受试者 1 个 repetition 作为校准数据。
  - [ ] 使用目标受试者剩余 repetitions 作为测试数据。
- [ ] 改变校准数据规模：
  - [ ] 1 个 repetition。
  - [ ] 2 个 repetitions。
  - [ ] 3 个 repetitions。
- [ ] 比较微调和特征对齐等策略。
- [ ] 绘制校准样本数量与性能之间的关系曲线。
- [ ] 在少样本自适应中比较稳定预训练模型与普通预训练模型。
  - [ ] 普通 CNN 预训练 + 微调。
  - [ ] Stable-CNN 预训练 + 微调。
  - [ ] 稳定重加权传统模型 + 少样本校准。

## 10. 基于稳定学习的跨个体改进方法

### 10.1 稳定学习问题建模

- [ ] 将训练数据表示为三元组：

```text
(x_i, y_i, e_i)
```

其中 `x_i` 是 sEMG 窗口或特征，`y_i` 是手势标签，`e_i` 是受试者环境 ID。

- [ ] 明确稳定学习目标。
  - [ ] 降低模型对个体特异性特征的依赖。
  - [ ] 保留与手势类别稳定相关的特征。
  - [ ] 提升未见受试者上的平均性能。
  - [ ] 降低不同测试受试者之间的性能波动。
- [ ] 决定主要方法路线。
  - [ ] 路线 A：稳定样本重加权 + 传统分类器。
  - [ ] 路线 B：稳定风险约束 + 深度模型。
  - [ ] 路线 C：稳定预训练 + 少样本微调。

### 10.2 稳定样本重加权

- [ ] 在传统特征空间中实现样本重加权。
- [ ] 在每个 LOSO fold 中只使用源域受试者学习权重。
- [ ] 设计样本权重目标。
  - [ ] 降低特征之间的冗余相关。
  - [ ] 降低特征与受试者 ID 的相关性。
  - [ ] 保持不同手势类别的有效样本量。
  - [ ] 避免少数样本权重过大。
- [ ] 实现类别条件稳定重加权。
  - [ ] 在每个手势类别内部，使特征分布尽量不依赖受试者 ID。
  - [ ] 避免把真正的手势差异错误消除。
- [ ] 使用加权样本训练传统分类器。
  - [ ] Weighted LDA。
  - [ ] Weighted SVM。
  - [ ] Weighted Random Forest。
- [ ] 对比普通训练和稳定重加权训练。

### 10.3 稳定风险最小化

- [ ] 在 1D-CNN 中实现按受试者分组的训练损失。
- [ ] 计算每个源域受试者的分类损失：

```text
Loss_s = CE(model(X_s), Y_s)
```

- [ ] 实现稳定风险目标：

```text
Loss = mean_s(Loss_s) + lambda * var_s(Loss_s)
```

- [ ] 调节 `lambda`，观察平均性能和跨受试者波动的变化。
- [ ] 对比普通 CNN 与 Stable-CNN。
- [ ] 记录每个训练受试者的 loss，检查模型是否只偏向少数受试者。

### 10.4 稳定表示学习

- [ ] 在 CNN encoder 输出的表示 `z` 上加入稳定约束。
- [ ] 实现特征去相关约束。
  - [ ] 降低表示维度之间的冗余相关。
  - [ ] 避免模型过度依赖个体特异性组合特征。
- [ ] 实现受试者不变约束。
  - [ ] 可选方案一：用域分类器预测受试者 ID，再通过梯度反转削弱受试者信息。
  - [ ] 可选方案二：用 MMD 或 CORAL 对齐不同受试者的表示分布。
- [ ] 实现类别条件对齐。
  - [ ] 在同一手势类别内部对齐不同受试者的表示。
  - [ ] 避免不同手势类别被过度混合。

### 10.5 稳定预训练与少样本微调

- [ ] 使用稳定学习目标在 39 名源域受试者上预训练模型。
- [ ] 使用目标受试者少量 repetition 微调。
- [ ] 比较普通预训练和稳定预训练的微调效果。
- [ ] 检查稳定预训练是否降低目标受试者所需校准样本量。

## 11. 其他跨个体改进方法对比

### 11.1 归一化策略消融

- [ ] 比较不归一化、源域全局归一化、源域逐通道归一化。
- [ ] 分析归一化是否减小个体间分布差异。

### 11.2 特征对齐

- [ ] 实现 CORAL loss 或 CORAL 特征变换。
- [ ] 实现 MMD loss，用于约束源域和目标域特征分布。
- [ ] 如果使用无标签目标受试者数据，需要明确该实验属于无监督域适应。
- [ ] 严格区分 source-only、无监督目标域适应和有监督少样本适应。

### 11.3 域对抗训练

- [ ] 实现 gradient reversal layer。
- [ ] 添加域分类器，用于区分受试者或区分源域和目标域。
- [ ] 训练特征提取器，使域分类器难以区分域。
- [ ] 比较 DANN 与 source-only CNN。

### 11.4 少样本微调

- [ ] 在源域受试者上预训练模型。
- [ ] 使用少量目标受试者 repetition 进行微调。
- [ ] 比较不同微调策略：
  - [ ] 只微调分类头。
  - [ ] 微调最后一个卷积块和分类头。
  - [ ] 用较小学习率微调整个网络。

## 17. 常见错误检查

- [ ] 跨个体实验中不能把所有受试者窗口随机混合后划分训练集和测试集。
- [ ] 零校准实验中不能在测试受试者上拟合归一化参数。
- [ ] 不能用目标测试受试者调超参数。
- [ ] 个体内实验中不能把同一个 repetition 的窗口同时放入训练集和测试集。
- [ ] 包含 rest 类时不能只报告 Accuracy。
- [ ] 比较不同方法时必须保证手势集合和数据划分一致。
- [ ] 不能使用测试集信息进行 transition window 或 outlier window 移除。
- [ ] 稳定学习的样本权重不能使用目标测试受试者标签估计。
- [ ] 如果使用目标受试者无标签数据，必须明确标注为无监督域适应设置。
- [ ] 风险方差约束只能在源域训练受试者上计算，不能把测试受试者加入训练目标。
